We are here for the better.

The Audio Working Group has published Web Audio API as a W3C Recommendation.
This specification describes a high-level Web API for creating, shaping,
and processing audio directly in a Web browser or in web applications.

With the API standardized and deployed as a royalty-free feature in Web browsers and other devices
and platforms, both on desktop and mobile, creating sound with the Web Audio API has become a
dependable, widely available, built-in capability, eliminating the need to install plugins or download separate applications.
